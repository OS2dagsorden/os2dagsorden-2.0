INTRODUCTION
------------
Poshy Tip is a jQuery Plugin for stylish tooltips.


INSTALLATION
------------
1. Download and unpack the Poshy Tip plugin in "sites/all/libraries".
   Link: http://vadikom.com/files/?file=poshytip/poshytip-1.1.zip
   The path must be like this "sites/all/libraries/poshy_tip/src/jquery.poshytip.min.js"
2. Place the module in your sites/all/modules/ directory.
3. Go to "Administer" -> "Site building" -> "Modules" and enable the module.


USAGE
-----
1. Go to admin/config/user-interface/poshy_tip.
2. Choose your CSS Selectors for trigger poshy_tip plugin.
3. Choose one of default themes or create your on theme.
4. Choose pages where poshy_tip plugin must be inserted.


DOCUMENTATION
-------------
1. Link: http://vadikom.com/demos/poshytip


POSHY TIP THEME
---------------
Yes, you can create new themes for Poshy Tip. Create and place your customized theme on 
sites/libraries/poshy_tip/src/theme-name and it will be automatic loaded.
